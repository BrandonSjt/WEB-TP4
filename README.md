# WEB-TP4

Proyek ini adalah website statis sederhana dengan dua halaman utama:
- Halaman login ([index.html](index.html))
- Halaman home berisi navbar, section home, collections, dan about ([home.html](home.html))

Semua tampilan diatur menggunakan satu file stylesheet, yaitu [style.css](style.css).

## Fitur Utama

- Form login sederhana dengan input username dan password
- Tombol submit yang mengarahkan pengguna ke halaman home
- Navbar responsif dengan menu hamburger untuk layar kecil
- Hero section dengan background image
- Gallery collections berbentuk grid
- Section about dengan foto profil dan deskripsi singkat
- Tata letak responsif untuk mobile dan desktop

## Struktur Proyek

- [index.html](index.html): Halaman login
- [home.html](home.html): Halaman utama setelah login
- [style.css](style.css): Seluruh styling untuk login page dan home page
- File gambar: aset visual untuk logo, hero background, koleksi, dan profil

## Cara Menjalankan

1. Buka folder proyek `WEB-TP4`.
2. Jalankan file [index.html](index.html) di browser.
3. Isi form login, lalu klik tombol submit.
4. Anda akan diarahkan ke halaman [home.html](home.html).

## Teknologi yang Digunakan

- HTML5
- CSS3

## Catatan

- Proyek ini belum menggunakan backend, jadi proses login belum melakukan validasi akun.
- Navigasi antar section di halaman home menggunakan anchor link (`#home`, `#collections`, `#about`).
- Untuk pengembangan lanjutan, Anda bisa menambahkan JavaScript untuk validasi form dan interaksi yang lebih dinamis.
